# Python List and Dictionary Programs

This repository contains simple Python programs demonstrating the use of dictionaries and list slicing.

## Task 1: Student Marks Using Dictionary
- Creates a dictionary with student names and their marks
- Takes a student's name as input
- Displays the corresponding marks
- Shows a message if the student is not found

## Task 2: List Slicing
- Creates a list of numbers from 1 to 10
- Extracts the first five elements
- Reverses the extracted elements
- Prints both the extracted list and the reversed list
